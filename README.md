### あそびかた・技術概要

#### あそびかた（おさかな呼び込みずんだもん版）
1. タイトル画面で `Score` にある曲（`.vvproj`）を選んで「スタート」します。  
2. 5秒カウントダウン後、お題に答えていきます。現在の実装では以下のお題が固定表示されます。  
   - ① やすいおさかなは？  
   - ② おいしいおさかなは？  
   - ③ しゅんのおさかなは？  
   - ④ すきなおさかなは？  
   - ⑤ いまたべたいおさかなは？  
3. たとえば「② おいしいおさかなは？」に対して魚の名前を入力して Enter を押すと、その回答が替え歌歌詞として処理されます。  
4. 5問ぶん入力が終わると、ずんだもん歌声の合成が走り、結果画面で曲が再生されます。  

入力ルール（現在実装）
- ひらがなのみ入力可
- 2〜4音節で入力
- 先頭の「ー」は不可
- 「っ」は不可
- 2〜3音節で入力した場合は、内部で4音節に補完して歌詞へ反映
- 各お題の制限時間は60秒（時間切れ時は自動補完）

#### 技術概要（現在実装）
- C++ / Siv3D / Visual Studio ベースの SceneManager 構成
- `WriteLyrics` シーンでお題表示と回答入力を処理し、回答を歌詞置換データとして保持
- `VocalSynthesis` シーンで `.vvproj` の歌詞を回答で差し替え、VOICEVOX API 経由で歌声を合成
- 歌唱キャラは ずんだもん（ノーマル, speaker ID: 3003）固定
- 合成した歌声 WAV と `Inst/<曲名>.mp3` を `Result` シーンで再生

#### 動作要件
- OS: Windows（配布済み exe 実行環境）
- 必須: VOICEVOX アプリを起動した状態（接続先デフォルト `http://localhost:50021`）
- 推奨 VOICEVOX バージョン: `0.25.0` 以降
- 同梱前提ファイル:
  - `Score/*.vvproj`（お題元・歌詞置換元）
  - `Inst/*.mp3`（伴奏）
  - `Texture/assets/*` などのリソース
- 画面は 1920x1080 前提で設計（モニタが小さい場合は自動でフルスクリーン化）
