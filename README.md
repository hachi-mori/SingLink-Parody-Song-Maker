## おさかな呼び込みずんだもん専用アプリ：シングリンク～おさかなバージョン～

とうほくプロコン2025 テーマ部門「魚好きを増やせ！」のエキシビション～OVER15作品プレゼンテーション～用に作成したアプリです。

**アプリ開発メンバー**
 - 💻プログラム：はちもり
 - 🎨デザイン：りょつ

### このアプリでできること
お題にあわせて魚の名前を入力すると、ずんだもんが「おさかな呼び込みソング」を歌ってくれるアプリです。  
「おいしいおさかなは？」のような問いに答えるだけで、簡単にオリジナルの曲が作れます。

### 遊ぶ前の準備
- Windows PC を用意する
- **[VOICEVOX](https://voicevox.hiroshiba.jp/)** アプリ（無料）をインストールして起動しておく
- **[リリース](https://github.com/hachi-mori/SingLink-Parody-Song-Maker/releases/tag/v1.0)** からダウンロードした zip を解凍し、exe を起動する

### あそびかた
1. タイトル画面であそぶ曲を選ぶ
2. 画面に出るお題に、魚の名前を入力して Enter
3. 入力した内容をもとに、ずんだもんの歌が生成されて再生される

### 入力のポイント
- ひらがなで入力してください
- できるだけ短い魚の名前だと入力しやすいです
- 1問ごとに制限時間があります
