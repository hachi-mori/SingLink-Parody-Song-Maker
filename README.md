## あそびかた➀：おさかなずんだもん

とうほくプロコン2025 テーマ部門「魚好きを増やせ！」のエキシビション～OVER15作品プレゼンテーション～用に作成しました。

### できること
お題にあわせて魚の名前を入力すると、ずんだもんが「オリジナルおさかなソング」を歌ってくれます。  
「おいしいおさかなは？」のような問いに答えるだけで、簡単にオリジナルの曲が作れます。

### あそびかた
1. タイトル画面であそぶ曲を選ぶ
2. 画面に出るお題に、魚の名前を入力して Enter
3. 入力した内容をもとに、ずんだもんの歌が生成されて再生される

### 入力のポイント
- ひらがなで入力してください
- できるだけ短い魚の名前だと入力しやすいです
- 1問ごとに制限時間があります

## あそびかた➁：おたんじょうびずんだもん

### できること
入力した名前やメッセージをもとに、ずんだもんが「おたんじょうびソング」を歌ってくれます。  
誕生日のお祝い用に、短い替え歌を手軽に作って再生できます。

### あそびかた
1. タイトル画面で「ハッピーバースデー」を選ぶ
2. 画面の案内にそって、お祝いしたい相手の名前を入力して Enter
3. 入力内容を反映した歌が生成され、伴奏つきで再生される

### 入力のポイント
- ひらがなで入力してください
- 名前は短めにするとテンポよくまとまります
