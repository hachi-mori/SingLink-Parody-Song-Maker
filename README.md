## おさかな呼び込みずんだもん専用アプリ：シングリンク～おさかなバージョン～

**アプリ開発メンバー**
 - 💻プログラム：はちもり
 - 🎨デザイン：りょつ

### アプリダウンロード
zipファイルをダウンロード後、解凍して、exeファイルを起動すると遊べます。  
※事前にVOICEVOXアプリのインストール、起動が必要です。  
https://github.com/hachi-mori/Tohoku-procon2025/releases/tag/v1.0

### このアプリでできること
お題にあわせて魚の名前を入力すると、ずんだもんが替え歌として歌ってくれるアプリです。  
「おいしいさかなは？」のような問いに答えるだけで、最後に1曲として再生されます。

### 遊ぶ前の準備
- Windows PC を用意する
- VOICEVOX アプリ（無料）をインストールして起動しておく
- ダウンロードした zip を解凍し、exe を起動する

### あそびかた（かんたんフロー）
1. タイトル画面で曲を選んでスタート
2. 画面に出るお題に、魚の名前を入力して Enter
3. これを数回くり返す
4. 入力した内容をもとに、ずんだもんの歌が生成されて再生される

### 入力のポイント
- ひらがなで入力してください
- できるだけ短い魚名だと入力しやすいです
- 1問ごとに制限時間があります
