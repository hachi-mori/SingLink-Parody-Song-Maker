# AGENTS instructions

このリポジトリに関する応答は原則としてすべて日本語で行ってください。

## 開発環境・ビルド

- 本プロジェクトは C++ / Siv3D / Visual Studio を前提とします。
- ビルド・実行確認は原則として Visual Studio から行ってください（`.sln` を使用）。
- 特別な理由がない限り、既存のソリューション構成・プロジェクト設定（Debug/Release、x64 など）を変更しないでください。

## C++ / Siv3D 実装ルール

- Siv3D 特有の記法や設計方針（`U""` 文字列、`String`、`Array`、`SceneManager` 運用など）は、まず既存実装を参照して合わせてください。
- 独自実装を追加する前に、同種の処理が既存コード（`Main.cpp`、`Common.hpp`、`Scene/*.cpp`）にないか確認し、命名・責務分割・描画/更新の流れを踏襲してください。
- 文字コードは既存ファイルに合わせて扱い、日本語テキストを壊さないように注意してください。
- アセットパス（`App/Texture`、`App/Inst`、`App/Score` など）は既存の相対パス構成を維持してください。

## 変更時の必須手順

コードやドキュメントを変更した場合は次の手順を守ってください：
1. `git status` で変更内容を確認し、`git commit` で保存すること。
2. テストやビルド手順が用意されている場合は実行を試み、その結果をPRの説明に含めること。
3. Pull Request を作成する際は `make_pr` を利用し、日本語で変更点の要約とテスト結果を記述してください。

より深い階層に別の `AGENTS.md` が存在する場合は、その指示を優先してください。
